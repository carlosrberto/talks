Front-End básico

  • HTML
  • CSS
  • JavaScript

HTML

Estruturar o conteúdo, definir se é um parágrafo,
uma lista, uma citação, etc.

As tags HTML por sí só não definem a aparência de nada.

CSS

Define a aparência que os elementos HTML terão

Define as cores de texto, cores de fundo,
dimensões, bordas, posição na tela e etc.

JavaScript

Da vida aos elementos HTML, possibilita que as coisas
aconteçam no Front-End, possibilita que o usuário interaja
com a tela e tenha respostas.

Elementos VS Componentes

Elementos: são os elementos do HTML.
Por exemplo p, div, span.

Componentes: Um ou mais elementos HTML agrupados
Geralmente um componente tem algum propósito, finalidade. Pode representar uma parte de um todo ou o próprio todo.

Quais horários existem entre a hora inicial e final?

Primeiro, sabendo que a hora inicial é 8:00 e o intervalo entre horários é 30 minutos, somamos 30 minutos a hora inicial até que seja menor ou igual a hora inicial do almoço.

Mas por que só até a hora do almoço?


8:00 + 0:30 = 8:30
8:30 + 0:30 = 9:00
9:00 + 0:30 = 9:30

11:00 + 0:30 = 11:30

Não podemos continuar aqui pois passaremos das 11:45
11:30 + 0:30 = 12:00

Chegando ao horário de almoço, a partir dele somamos o valor do intervalo (30 minutos) até seja menor ou igual ao horário final de almoço.


11:45 + 0:30 = 12:15
12:15 + 0:30 = 12:45
12:45 + 0:30 = 13:15

Não podemos continuar aqui pois passaremos das 13:20
13:15 + 0:30 = 13:45

Continuamos a partir do horário final do almoço


13:20 + 0:30 = 13:50
13:50 + 0:30 = 14:20
...
17:20 + 0:30 = 17:50



Como o horário final é 18:00, passamos no máximo mais um intervalo


17:50 + 0:30 = 18:20

E assim temos a lista com todos os horários que devem aparecer:


08:00, 08:30, 09:00, 09:30
10:00, 10:30, 11:00, 11:30
11:45, 12:15, 12:45, 13:15
13:20, 13:50, 14:20, 14:50
15:20, 15:50, 16:20, 16:50
17:20, 17:50, 18:20

Traduzindo para uma pseudo-linguagem, serial algo como:


ENQUANTO: "a hora inicial for menor que a hora final"
    SE: "a hora inicial estiver entre o horário inicial e final de almoço"
        SALVE: "a hora inicial do almoço na lista"

        ENQUANTO: "a hora inicial for menor que a hora final do almoço"
            SOME: "hora inicial + 0:30"
            SALVE: "o resultado na LISTA"

        SALVE: "a hora final de almoço na lista"
    SENÃO:
        SOME: "hora inicial + 0:30"
        SALVE: "o resultado na LISTA"

Com essa lista de horas disponível, "dizemos" para a linguagem:


PARA_CADA: "hora na lista de horas"
    RENDERIZE: "uma coluna"
